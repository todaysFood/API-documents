    Method: GET
    URL: /api/feed/{post_id:int}/

    • Request
      • None

    • Response

      • Normal 
        HTTP 200 OK
Content-Type: application/json
{
  "id": 1,
  "title": "글 제목",
  "pub_date": "{timezone_now}",
  "modified_date": "{수정일자, default=Null}",
  "content": "글 내용",
  "rating": "1.0",
  "place_id": 1500,
  "author": "{user.UUID}"
}

    Method: PUT(수정)

    • Request

      • Header 
        {
"Authorization : Bearer {JWT TOKEN VALUE}"
}
      • Body 
          {
  "title" : "제목 수정",
  "rating" : 1.0,
  "place_id" : 1500,
  "content" : "수정 테스트"
  }

    • Response

    • Normal (status : 200 OK) 
      "Post edited"
    • 글쓴이와 User가 일치하지 않을 때 (status : 403 Forbidden) 
      {
"detail": "이 작업을 수행할 권한(permission)이 없습니다."
}
    • 인증정보가 유효하지 않을 때 (status : 401 Unauthorized) 
      {
"detail": "Given token not valid for any token type",
"code": "token_not_valid",
"messages": [
    {
        "token_class": "AccessToken",
        "token_type": "access",
        "message": "Token is invalid or expired"
    }
]
}

    Method: DELETE(삭제)
    - Request - Header 

    {
"Authorization : Bearer {JWT TOKEN VALUE}"
}

    • Response
    • Normal (status : 204 No content) 
      "Post deleted"
    • 글쓴이와 User가 일치하지 않을 때 (status : 403 Forbidden) 
      {
"detail": "이 작업을 수행할 권한(permission)이 없습니다."
}
    • 인증정보가 유효하지 않을 때 (status : 401 Unauthorized) 
      {
"detail": "Given token not valid for any token type",
"code": "token_not_valid",
"messages": [
    {
        "token_class": "AccessToken",
        "token_type": "access",
        "message": "Token is invalid or expired"
    }
]
}
